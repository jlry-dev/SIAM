# KPIs - Bit Battleship

## Objective
Measure system reliability, gameplay performance, and backend stability.

---

# Key Performance Indicators

| KPI | Description | Target |
|-----|-------------|--------|
| Server Uptime | Percentage of server availability | 99% |
| Matchmaking Success Rate | Successful room pairings | 95% |
| Average Match Start Time | Time before game starts | < 5 seconds |
| API Response Time | Backend response speed | < 200ms |
| Active Concurrent Rooms | Number of active game rooms | Monitor growth |
