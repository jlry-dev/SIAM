# Cost–Benefit Analysis - Bit Battleship

## 1. Cost Estimation Model

This project uses a simple student-based estimation model:
- Development time cost (student hours)
- Infrastructure cost (server + database)
- Maintenance cost (debugging + updates)

---

## 2. Development Cost

Assumptions:
- 4 developers
- 2 hours/day
- 10 days sprint
- ₱150/hour (estimated student dev rate)

Calculation:

4 developers × 2 hours/day × 10 days = 80 hours  
80 hours × ₱150 = ₱12,000

**Total Development Cost: ₱12,000**

---

## 3. Operational Cost

| Item | Cost Estimate |
|------|--------------|
| Koyeb hosting (basic plan) | ₱500/month |
| PostgreSQL database | ₱300/month |
| Domain (optional) | ₱100/month |

**Total Monthly Operational Cost: ₱900/month**

---

---

## 5. Intangible Benefits

- Software engineering workflow exposure

---

## 6. ROI Calculation

Assume educational value equivalent:

- Skill gain value: ₱30,000 (estimated market learning value)
- Total cost (1 month): ₱12,900

ROI formula:

ROI = (Benefit - Cost) / Cost × 100

ROI = (30,000 - 12,900) / 12,900 × 100  
ROI ≈ 132.56%

---

The project is highly beneficial despite operational and development costs.

- Consider expanding into full real-time game platform

## Conclusion

Bit Battleship provides strong educational value and practical software engineering experience with a positive return on investment.
